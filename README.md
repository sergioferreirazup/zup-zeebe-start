# zup-zeebe-start
### Projeto base para utilização de Zeebe com Spring Boot
---
## Componentes/Funcionalidades

- Web Api
- Flyway
- Postgres Datasource
- Zeebe Configuration
- Zeebe Deployment
- Zeebe Starter
- Zeebe Worker
---
## Pré-requisitos
- Java 11
- Maven
- Docker/-compose
---
## Inicialização da Infraestrutura

```shell
$ cd infrastructure
$ docker-compose up -d
```
